# dev-tools

Developer workflow utilities for Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| `/dev-tools:issue` | Create well-structured GitHub issues |
| `/dev-tools:docs` | Write technical documents (Toss writing principles) |
| `/dev-tools:project-log` | Log findings and decisions to project notes |

## Prerequisites

- `gh` CLI (for issue skill): `brew install gh && gh auth login`

## Install

```bash
claude plugin add github:GoBeromsu/claude-code-plugins --path plugins/dev-tools
```
