# development

Developer workflow utilities for Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| `/development:issue` | Create well-structured GitHub issues |
| `/development:docs` | Write technical documents (Toss writing principles) |
| `/development:project-log` | Log findings and decisions to project notes |

## Prerequisites

- `gh` CLI (for issue skill): `brew install gh && gh auth login`

## Install

```bash
claude plugin add github:GoBeromsu/claude-code-plugins --path plugins/development
```
