# thinking-tools

Decision-making and analysis frameworks for Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| `/thinking-tools:antifragile` | Apply Taleb's Barbell Strategy to decisions |
| `/thinking-tools:5whys` | Interactive root-cause analysis (5+ rounds) |
| `/thinking-tools:first-principles` | Decompose problems to irreducible fundamentals |
| `/thinking-tools:interview` | Probe plans for blind spots before implementation |

## Install

```bash
claude plugin add github:beomsu/claude-code-plugins --path plugins/thinking-tools
```
